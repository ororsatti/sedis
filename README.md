# Sedis - (S)imple R(edis)

A programming practice to learn TCP and better learn C.

## Architecture:
1. Simple TCP server.
2. RESP parser.
3. Map as simple cache.


## Supports: 
  - `PING`
  - `SET <key> <value>`
  - `GET <key>`

## Planning to supprt:
  - `DEL <key1> <key2> ...`
  - `SCAN <cursor> <size>`
  - `KEYS <pattern>`
